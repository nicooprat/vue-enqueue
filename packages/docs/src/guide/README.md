# Getting Started

## Installation

Install the `vue-enqueue` package:

```sh
npm i vue-enqueue
```

Or

```sh
yarn add vue-enqueue
```

## Quick start

Import the `Enqueue` component from `vue-enqueue`:

```vue{2,6}
<script>
import { Enqueue } from 'vue-enqueue'

export default {
  components: {
    Enqueue,
  },
}
</script>
```
